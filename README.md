# AI Conversational Chatbot

A chatbot developed using Python and Jupyter Notebook by integrating Groq API with Meta Llama 3.3 language model.

## Features

- Conversational AI interaction
- Context memory support
- Real-time response generation
- Natural language processing

## Technologies Used

- Python
- Jupyter Notebook
- Groq API
- Meta Llama 3.3 70B Model
- OpenAI Python Library

## Workflow

User Input  
↓  
Python Program  
↓  
Groq API Request  
↓  
Llama Model Processing  
↓  
Response Generation  
↓  
Output to User

## Installation

Install dependencies:

pip install openai

Run the notebook or python script.

## Example

You: What is Python?  
Bot: Python is a high-level programming language used for software development.
